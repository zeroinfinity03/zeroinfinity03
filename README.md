# Surya Singh

**AI/ML Engineer** — I build production machine learning and LLM systems that sit close to real decisions.

My professional focus is applied ML in high-stakes domains: fraud and risk models, real-time scoring, graph-based detection, retrieval-augmented GenAI systems, and the infrastructure to deploy, monitor, and continuously improve them. Alongside that, I build practical LLM tooling — RAG applications, agentic workflows, local inference, and workflow automation — with a bias toward systems people can actually use.

**Focus areas:** Production ML · LLM Applications · RAG · Agentic Workflows · Local Inference · MLOps

## Featured Projects

### KnowFlow — Multimodal Local RAG Assistant
`FastAPI` · `Llama 3.2 (4-bit)` · `ChromaDB` · `Playwright` · `Hugging Face` · `GCP TTS`

A fully local multimodal AI assistant built on quantized Llama 3.2, with a ChromaDB-backed RAG pipeline supporting document Q&A across six file formats and text, image, and voice interaction.
[Repository](https://github.com/zeroinfinity03/knowflow-beta1.0)

### GenAI Risk Intelligence System
`GPT-4` · `LLaMA` · `LangChain` · `ChromaDB` · `FAISS` · `FastAPI` · `Docker` · `AWS` · `MLflow`

A modular GenAI assistant with RAG over 500+ financial and policy documents, producing audit-ready risk summaries. Deployed as a scalable FastAPI microservice with WebSocket streaming and MLflow experiment tracking.
[Repository](https://github.com/zeroinfinity03)

### Algospeak-Aware Content Moderation
`FastAPI` · `Qwen2.5-3B` · `QLoRA` · `Polars` · `Next.js`

A two-stage moderation system pairing a hot-reload JSON algospeak normalizer (100+ variants — character swaps, phonetics, euphemisms) with a QLoRA-tuned Qwen2.5-3B classifier fine-tuned on a 52K-sample set derived from the Jigsaw toxicity dataset.
[Repository](https://github.com/zeroinfinity03/Algospeak-Aware-Content-Moderation)

### SQLite3 MCP Server
`FastAPI` · `FastMCP` · `MCP SDK` · `Gemini 2.0 Flash` · `SQLite3`

A local Model Context Protocol server exposing eight read-only database tools, letting LLMs inspect schemas and run safe SQL queries. Integrated with Gemini 2.0 Flash via the MCP Python SDK for natural-language database exploration.
[Repository](https://github.com/zeroinfinity03/sqlite3-mcpserver)

### Credit Card Fraud Detection
`Python` · `scikit-learn` · `XGBoost` · `CatBoost` · `SMOTE`

Ensemble models over 284K+ transactions to detect rare fraud (~0.17% positive class), using SMOTE to handle severe class imbalance and threshold tuning to maximize recall while holding a stable ROC-AUC.
[Repository](https://github.com/zeroinfinity03/credit-card-fault_detection)

### Customer Churn Prediction
`Python` · `scikit-learn` · `Random Forest` · `Pandas`

A churn classifier reaching 86% accuracy and outperforming logistic-regression baselines, with engineered features (e.g. collect-to-redeem ratio) and feature-importance analysis to surface the behaviors driving churn.
[Repository](https://github.com/zeroinfinity03/customer_churn_prediction)

### Breast Cancer Classification
`Python` · `TensorFlow` · `Keras` · `scikit-learn`

A deep-learning classifier on the Wisconsin Breast Cancer dataset reaching 93.86% accuracy, using standardization and correlation analysis to clean features and reduce false negatives.
[Repository](https://github.com/zeroinfinity03/Breastcancer)

### Movie Recommendation System
`Python` · `Streamlit` · `TMDB API` · `scikit-learn`

A content-based recommender using TF-IDF vectors and cosine similarity over combined text features (overview, genres, keywords, cast, crew), deployed as a Streamlit app with live TMDB posters.
[Repository](https://github.com/zeroinfinity03/MoviesRecommendation)
